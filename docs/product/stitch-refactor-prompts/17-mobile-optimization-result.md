# Stitch Prompt: Mobile Optimization Result

Create mobile optimization result screens.

Use `00-system-summary.md` and `00-design-system.md`.

Required content:
- estimate total;
- savings;
- mode;
- location/radius status;
- item decision cards;
- selected variant;
- store;
- price;
- trust factor;
- evidence count;
- freshness;
- distance only when precise coordinates exist;
- actions: open checklist, report price, send receipt.

Interaction:
- decision cards expandable;
- one primary CTA at bottom;
- no raw debug codes.
