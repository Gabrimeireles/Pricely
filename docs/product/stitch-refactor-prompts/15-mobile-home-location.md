# Stitch Prompt: Mobile Home and Location

Create mobile app home/location screens for Pricely.

Use `00-system-summary.md` and `00-design-system.md`.

Screen purpose:
Mobile is shopper-first. It should help the user continue buying, optimize a list, and submit receipts.

Required content:
- Current city and location precision card.
- States:
  - city-only;
  - precise coordinates;
  - CEP fallback;
  - denied/restricted;
  - unsupported.
- Next action:
  - continue checklist;
  - optimize list;
  - create list;
  - scan receipt QR.
- Saved lists preview.
- Reward status preview.

Mobile rules:
- Bottom navigation.
- Bottom sticky primary action.
- Large touch targets.
- No dense admin content.
