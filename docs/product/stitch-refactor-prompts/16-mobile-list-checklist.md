# Stitch Prompt: Mobile List and Checklist

Create mobile list editor and checklist screens.

Use `00-system-summary.md` and `00-design-system.md`.

List editor:
- fast product search;
- quantity stepper;
- variant intent: any variant vs fixed variant;
- list items with image and selected intent;
- sticky save/optimize action.

Checklist:
- store grouping;
- item checkbox;
- selected variant and expected price;
- report wrong price;
- completion prompt;
- optional total paid;
- CTA to scan/send receipt.

Mobile constraints:
- thumb-friendly;
- minimal typing during shopping;
- bottom sheet for details/report.
